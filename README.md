# mi_biblioteca
Plugin sencillo para gestionar una colección de libros. Taller realizado en la meetup de WordPress Collado Villalba

Descripción de los ficheros y la estructura en árbol.
Se ha usado en la programación el MVC(Módelo Vista-Controlador)

- **mi_biblioteca**
- **mi_biblioteca**\mi_biblioteca.php
- **mi_biblioteca\assets**
- **mi_biblioteca\assets**\mi_biblioteca.css   (Hoja de estilos del plugin)
- **mi_biblioteca\assets**\mi_biblioteca.js    (Código JavaScript necesario del plugin)
- **mi_biblioteca\inc**
- **mi_biblioteca\inc**\controlador.php   (Funciones que interactuan con la base de datos CONTROLADOR)
- **mi_biblioteca\inc**\views.php         (Las VISTAS necesarias para que se pueda administrar el plugin correctamente)
